# 제 1회 현대 해커톤
## 프로젝트 명 : 차량 내부 감지 시스템
## 팀명 : SaveUs (우리의 삶에 안전을 더해 드립니다.)
## 개발 환경 
* Raspbian 4.4 
* Android Studio
* Spring Framework 4.0
* Tomcat8.0
* Maven
* JDK8.0
* Amazon Web Service
* python2.7

## 프로젝트 설명
* 시동이 꺼진 차량의 내부에서 움직임이 포착되면 사진 촬영 후 운전자에게 전송해줍니다.
* 차량이 주차된 근처에 경찰서를 찾아주어 연락을 가능하게 해줍니다.
